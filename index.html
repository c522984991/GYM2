三分化训练（周三天）— 每日必做：龙门吊卷腹·引体卷腹·肩 前/中/后

设备：弹力带、9kg 哑铃、瑜伽垫、普通椅子。每训练日包含：必做三项 + 额外 3 个针对当日目标部位的动作（胸/背/腿）。基于三分化训练的常见实践与分化方案建议。[三分化方案参考](https://www.familydoctor.cn/article/jianshen-fenhua-jihua-xuanze-3182.html){target="_blank" class="gpt-web-url"}、[什么叫三分化和五分化说明](https://yinzhipeng123.github.io/markdown_log/soft/%E4%BB%80%E4%B9%88%E5%8F%AB%E4%B8%89%E5%88%86%E5%8C%96%E5%92%8C%E4%BA%94%E5%88%86%E5%8C%96/){target="_blank" class="gpt-web-url"}

总原则（简短）
  • 每次训练热身 8–10 分钟（肩袖激活 + 动态热身）。
  • 每天必须包含：龙门吊卷腹（悬挂卷腹）、引体卷腹（悬腿/悬举+卷腹）、肩部三束（前/中/后）练习。
  • 剩余 3 个动作针对当天主部位（Day1 胸 / Day2 背(含引体向上) / Day3 腿）。
  • 训练参数建议（可调）：复合动作为 3–4×6–10，辅助为 3×8–15，肩袖外旋为 2–3×12–20。
热身（每次）
  • 动态热身 3–4 分钟（原地高抬膝、臂绕环、躯干扭转）。
  • 肩袖激活（弹力带外旋 2×12、内旋 2×12、Band pull‑aparts 2×15）。
  • 下肢唤醒：臀桥 ×10、空蹲 ×10。
每日「必须三项」说明（每天都做）
龙门吊卷腹（悬挂卷腹）
参数：3 组 × 8–15 次（或 3×30s）。要点：用腹部卷起胸向髋方向，避免以臀摆或摇摆借力。若无法悬挂可做仰卧举腿+仰卧卷腹替代。
引体卷腹（悬腿 + 卷腹）
参数：3 组 × 8–12 次。要点：结合髋屈（抬腿）与脊柱卷曲，强调控制与呼吸。若无法悬挂用椅子支撑的悬腿或平地仰卧举腿替代。
肩部三束（前 / 中 / 后） — 每日常规
做法示例（可超组）：前束：哑铃前平举 2×10（或带）；中束：弹力绳侧平举 3×12–15；后束：弹力带面拉或俯身反向飞 3×12–15。每次训练至少做一组轻量肩袖外旋（2×15）。
周日程（建议：三天）
  • Day A（胸日）：龙门吊卷腹、引体卷腹、肩三束 + 弹力带平躺胸推 / 地面哑铃卧推 / 俯卧撑（3 个针对胸）
  • Day B（背日）：龙门吊卷腹、引体卷腹、肩三束 + 引体向上（主拉）/ 单臂哑铃划船 / 弹力带划船（包含引体向上）
  • Day C（腿日）：龙门吊卷腹、引体卷腹、肩三束 + 哑铃高举深蹲 / 保加利亚分腿蹲 / 臀桥（3 条腿相关动作）
示例：每日日程（单次训练 45–55 分钟）
  1. 热身 8–10 分钟（动态 + 肩袖激活）。
  2. 必做三项（悬挂卷腹 / 引体卷腹 / 肩前中后）按上方参数完成（合计约 20–25 分钟）。
  3. 目标部位 3 个动作（复合动作先，辅助后），按 3×8–12 或 4×6–10 完成（约 20 分钟）。
  4. 拉伸与恢复 3–5 分钟（肩/胸/背/腿的针对拉伸）。
具体动作清单（含替代与要点）
胸（Day A 的 3 个胸动作示例）
弹力带平躺胸推
3–4×8–12；要点：背部稳定，胸肌收缩，避免肩代偿。替代：地面哑铃卧推 3×6–10 或 俯卧撑 3×8–15。
弹力带飞鸟 / 俯身飞鸟
3×10–15，控制离心，增强胸外展与线条。
俯卧撑（进阶：脚抬高 / 负重）
3×8–15，视体重与能力调整。作为胸部耐力/体型动作。
背（Day B 的 3 个背动作示例）
引体向上 / 弹力带助力
4组 尽力（目标 6–12 次/组）。若无法完成，做椅子反向划船或弹力带助力。
单臂哑铃划船（椅子支撑）
4×8–12/侧；慢速离心并在顶点挤肩胛。替代：弹力带划船 3×10–15。
弹力带面拉（Face‑pull） / 反向飞鸟
3–4×12–15；重点做外旋并收缩后肩/肩胛，保护肩袖并改善姿态。
腿（Day C 的 3 个腿动作示例）
哑铃高举深蹲（Goblet squat）
3–4×8–15；若 9kg 太轻，做慢速离心或增加组数/做单腿变体。
保加利亚分腿蹲（椅子支撑）
3×8–12/侧；高强度单腿练习，能提升腿部与稳定性。
臀桥 / 臀推（可夹哑铃）
3–4×10–15；顶端收缩与臀中肌触发。
进阶与注意事项
  • 若 9kg 哑铃不够重，可做单侧或慢速离心（3–4 秒下放）来提升张力。
  • 肩袖外旋应每日高频低负荷（练前与训练后），防止肩伤并改进肩外旋肌群。
  • 保持饮食与恢复（三分之一为优先）：蛋白质 1.6–2.2 g/kg，充足睡眠与 NEAT。
  • 如出现疼痛（非正常肌肉酸痛），停止并检查动作/咨询专业人士。
